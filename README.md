# Movitor COVID-19

## Installation

```bash
npm install

or

yarn install
```

## Run

```bash
npm run start

or 

yarn start
```

### Technologies used

* [RN] - React Native
* [Axios] - Axios
* [ReactNavigation] - React Navigation v5.
* [ReactNavigationModal] - React Native Modal
* [Redux] - Redux
* [ReduxPersist] - Redux Persist
